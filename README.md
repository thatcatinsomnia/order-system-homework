# 餐點系統
點餐系統前端練習

# How to run
1. git clone https://github.com/thatcatinsomnia/order-system-homework.git
2. cd order-system-homework
3. pnpm i
4. pnpm watch:json (模擬 server 回傳 api 資料)
5. pnpm dev (前端 server)


## 需求 / 限制
- 餐點欄位必須包含以下項目:
   - Name
   - Price
   - Notes
   - Customer
- 餐點資料使用 LocalStorage 儲存
- 要有 Responsive design
- UI/UX 盡量好一點
- CSS 使用 OOCSS 命名方式

## 功能
- [ ] 餐點列表
- [ ] 新增餐點
- [ ] 修改餐點
- [ ] 移除餐點
   - [ ] alert 刪除提示
